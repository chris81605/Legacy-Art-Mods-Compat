# 舊版圖片名稱適配（增強版）

基於原作者版本進行擴充。

## 原作者功能

- 相容舊版圖片命名規則
- 自動轉換新版／舊版部分圖片路徑
- 修正部分舊版服裝資料相容問題

## 本版本額外增強

### 手持物圖片相容增強

補強新版與舊版手持物圖片命名規則的相容性。

可能改善部分舊版手持物於新版中的顯示問題，並修正部分新版命名無法正確對應至舊版素材的情況。

### 灰階圖片優先載入

調整圖片搜尋優先順序。

當同時存在：

- `xxx_gray.png`
- `xxx.png`

時，將優先載入 `xxx_gray.png`，以符合舊版素材的預期顯示效果。

---
### 通配符路徑相容規則

新增可使用通配符 (`*`) 的圖片路徑相容規則。

可針對特定路徑設定：

- **Fallback**：新版圖片不存在時，自動改用舊版路徑。
- **Override**：優先使用指定舊版路徑，可用於覆蓋新版原圖或美化素材。

支援一組規則對應多個候選路徑，以提升不同第三方模組之間的相容性。
